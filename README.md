# MusicApp

Application for listening to music with Angular based on [SpotifyApi](https://developer.spotify.com/documentation/web-api/)
<br />
![image](https://user-images.githubusercontent.com/29175500/164913346-e1daffb5-42b2-41ae-b9de-9bba415d8fc5.png)

Please remember use this application you need to add yours spotify cliend id and client secret key. Add it to environments/environment.dev.ts

### Technologies
+ Angular
+ NgRx
+ RxJs
+ TypeScript
+ i18n library - [ngx-translate](https://www.npmjs.com/package/@ngx-translate/core)
